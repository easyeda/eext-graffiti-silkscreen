# Graffiti Silkscreen

[中文](./README.md)

## Feature Overview

Graffiti Silkscreen is a simple and easy-to-use color silkscreen design tool designed for users, suitable for designing simple logos and background elements.

## User Interface

Users can access the following functions through the top menu bar:
- Silkscreen > Graffiti Silkscreen

## Usage

1. **Draw a sketch**: Draw a sketch of the desired material on the graffiti canvas. Supports clearing the canvas, undoing operations, and saving graffiti.
2. **Select image style**: Enter a prompt in the input box and select the desired image style.
3. **Configure API key**: Enter your Tongyi Key in the bottom input box.
4. **Generate and download materials**: Click the upload button, the plugin will automatically query the results, and after the materials are generated, they will be automatically downloaded and placed on the canvas.

Please note that when using this feature, you need to enable "External Interaction" in the Extension Manager.

**For detailed tutorials, please visit**: [Tutorial Documentation](https://lceda001.feishu.cn/wiki/DiHQw2gcyitmcYkeQLwcPNsvnw0)
